#### 악성댓글 자동수집 프로그램.

## 환경설정

* Python 웹 크롤링을 하기 위한 환경설정.
```
 
  pip install bs4
  
  pip install requests
 
```

* selenium 설치.
```
 pip install selenium

```
selenium은 여러 언어를 지원하는 웹 자동화 라이브러리 이다.

* chrome driver 설치
 (chrome 버전 확인 후 설치.)
 
```
  https://sites.google.com/a/chromium.org/chromedriver/downloads

```
